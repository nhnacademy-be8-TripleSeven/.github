## 주요 기능

### 인프라
- AWS 또는 기타 클라우드 서비스 환경 설명

### 주소
- 주소 관련 CRUD 기능

### 저자
- 저자 관련 데이터 관리 및 연계 기능

### 도서
- 도서 등록, 수정, 삭제, 검색 기능

### 카테고리
- 카테고리별 도서 분류 및 필터링

### 장바구니
- 사용자 장바구니 관리 및 결제 연동

### 쿠폰
- 쿠폰 발급 및 적용 기능

### 회원
- 회원 가입, 로그인, 프로필 관리

### 주문
- 주문 생성, 조회, 상태 업데이트

### 결제
- 결제 시스템 연동 및 처리

### 포인트
- 포인트 적립 및 사용 기능

### 리뷰
- 리뷰 작성, 조회, 수정, 삭제

### 태그
- 도서 태그 관리 및 검색 기능
