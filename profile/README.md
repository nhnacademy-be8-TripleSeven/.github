## 🛠️ Service Introduce

- **NHN24 Bookstore**는 클라우드 환경을 적용한 도서 쇼핑몰 웹 애플리케이션입니다.
- 주요 기능:
  - **회원 관리**: 회원가입, 로그인, 권한 설정
  - **도서 검색 및 관리**: 도서 등록, 검색, 분류, 리뷰
  - **주문 및 결제**: 장바구니, 주문 생성 및 상태 확인
  - **포인트 및 쿠폰**: 적립, 쿠폰 관리 및 적용
